## ðŸ“¥ Installation & Setup

### Option A: From .ZIP Archive
If you received the game as a `.zip` file, follow these steps to play:

1. **Extract the Archive**
   - Right-click the `.zip` file and select "Extract All" (or use WinRAR/7-Zip).
   - Open the extracted folder.

2. **Install Node.js**
   - Ensure you have **Node.js** installed on your computer.
   - Download it here: [nodejs.org](https://nodejs.org/) (LTS version recommended).

3. **Open Terminal**
   - Inside the game folder, right-click in an empty space and select **"Open in Terminal"** (or Git Bash / PowerShell).
   - Alternatively, press `Win + R`, type `cmd`, and use `cd` to navigate to the folder.

4. **Install Dependencies**
   Run the following command to install all necessary libraries:
   ```bash
   npm install
   ```

5. **Start the Game**
   Launch the local development server:
   ```bash
   npm run dev
   ```
   - The terminal will show a local URL (usually `http://localhost:5173` or `http://localhost:3000`).
   - Open that link in your browser (Chrome/Edge recommended).

## ðŸŒŸ Features

- **Infinite Procedural World**: Every playthrough offers a unique terrain generated by Simplex Noise.
- **Dynamic Threat System**:
  - **Meteor Showers**: Radar warning system with impact zones.
  - **Enemies**: Standard Zombies and fast-moving **Crawlers (Spiders)** on higher difficulties.
- **Survival Mechanics**: Manage **Health** and **Oxygen** levels.
- **4 Difficulty Levels**:
  - **Easy**: For exploration.
  - **Medium**: Balanced survival.
  - **Hard**: Scarce resources, dangerous enemies.
  - **Survivor (Severe)**: The ultimate test. Fast spiders, constant meteors.
- **Training Mode**: Practice your movement in a safe environment.
